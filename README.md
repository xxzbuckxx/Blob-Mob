# Blob-Mob
Javascript Canvas Game

**Version 10.5**

Blobmob is an html5 Canvas game that was created during 2016 for coding practice.

Blobmob was originally designed on a chromebook webstie HTML compilier but was moved to a github respitory in its final stages. All previous saves are owned and available in google drive if needed; music and sound effects made on garageband; sound utilizing howler.js.
